Range slider plugin for JQuery.

Extends JQuery class with .initSlider() method.
Include it after JQuery.js is included.

How to use: 

1. Create <div class=«container_name»> element to be transformed into Slider container;
2. Call $(«.container_name»).initSlider(args);

	args: range: {min: x, max: y} - range of values;
		flag: «string_flag» - necessary for other plugins, requiring rangeSlider.js;

Creating simple slider is going to be like that:

	$(«.sliderBody»).initSlider(range: {min: 0, max: 100}, flag: «someflag»});	

3. div.sliderGap and p.sliderValue are generated by plugin. By click and drag this Gap, slider is changing it’s value; p.sliderValue presents current value. If necessary, you can remove it (rangeslider.js: 3) or display: none in css to hide concrete element.

4. Add necessary styles: 

.container_name{
	position: relative;
}

.sliderGap{
	position: absolute;
	top: -this.width/2;
}

Those are necessary styles. You can style you’r slider however you wish;

5. Enjoy;
